# wenjobstatus

马来西亚高校教职求职状态公开看板。

这个仓库用于追踪求职投递、筛选、面试和 offer 相关状态。公开页面只展示学校/机构、岗位、地点、平台、投递方向和公开状态，不放详细住址、签证表单答案、简历文件名、求职信文件名或内部备注。

网页入口：

`https://wenquan0816.github.io/wenjobstatus/`

## 当前数据

- 已投递岗位：17
- 已覆盖机构：11
- 重复职位记录：2
- 已筛除岗位：1

公开页面只保留已投递、重复发布和硬性条件不符记录；待遇/薪资列只记录招聘信息中公开的待遇，未公开的岗位标为“招聘信息未公开/待核实”。

## 数据来源

公开数据由本地求职申请记录过滤生成到 `applications.json`。

后续可继续加入这些状态：

- 面试邀请
- 面试完成
- 等待结果
- offer
- rejected
- withdrawn
