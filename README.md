# wenjobstatus

马来西亚高校建筑学与室内设计教职求职状态公开看板。

这个仓库用于追踪建筑学、建筑设计、室内设计、室内建筑、建成环境与可持续发展相关高校教职投递、筛选、面试和 offer 状态。公开页面只展示核心对口岗位的学校/机构、岗位、地点、平台、投递方向和公开状态，不放详细住址、签证表单答案、简历文件名、求职信文件名或内部备注。

网页入口：

`https://wenquan0816.github.io/wenjobstatus/`

## 当前数据

- 已投递核心对口岗位：7
- 已覆盖机构：5
- 重复职位记录：0
- 已筛除岗位：0

公开页面只保留建筑学、室内设计、室内建筑、建成环境和可持续发展相关方向的核心对口岗位；待遇/薪资列只记录招聘信息中公开的待遇，未公开的岗位标为“招聘信息未公开/待核实”。泛艺术设计、动画、传媒、时装、纯 IT/AI、商科、医学、语言等非核心方向不列入本公开看板。

## 数据来源

公开数据由本地求职申请记录过滤生成到 `applications.json`。

后续可继续加入这些状态：

- 面试邀请
- 面试完成
- 等待结果
- offer
- 不太可能推进 / rejected
- withdrawn
